
# Cosmic Portfolio - The Power of Creation on the Fingertips

A portfolio to showcase my skills, projects, and experience as a web developer, with a cosmic theme inspired by creativity and power.

## Tech Stack

- **Next.js**: Framework for building the React application.
- **GSAP**: JavaScript animation library for powerful, customizable animations.

## Installation

1. Clone this repository:
   ```bash
   git clone [https://github.com/AnonX9/cosmic-portfolio](https://github.com/AnonX9/cosmic-portfolio).git
   cd cosmic-portfolio)
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Project Structure

- **`components`**: Contains React components for each section (Hero, About, Projects, Skills, Contact, Footer).
- **`public`**: Contains static assets such as images and icons.
- **`styles`**: Contains global styles and CSS modules for component styling.

## Animations

Using GSAP for smooth, custom animations across sections, including scroll-triggered animations and interactive effects.
